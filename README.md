# 2048
这是一个使用SDL2图形API制造的2048游戏，美术资产取自Minecraft。它是我的第一个游戏项目，当年没有大模型的指导，纯手搓，代码水平一般。希望能有供各位交流的地方，更多希望能得到各位的指点。复现的时候记得部署SDL2哦！
